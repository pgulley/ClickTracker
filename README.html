README:

The Mouse tracker (MT) and Data processor (DP) both go together.


Installation:

First, if not already present, install Python-2.7.2 using the included installer. The suggested settings should work for your needs.

After installing python, install both pyHook and pywin32- these shouldn't take long to run. 


1. Before using either of the scripts, open them in a simple text editor (Notepad, for example. Word may mess things up), and use the Find-Replace function to replace all the occurrences of "K:" with the full location name of whatever folder you want output stored in (For example: "C:/users/name/desktop/output").


2. After that, right click on "clicktracker.py" and choose "open with… Python". This will run the data collection script. Follow the instructions on the screen to collect data.


3.After you have collected your data, run 'datachecker.py' as you ran click tracker. This will allow you to determine if the clicks which have been recorded occurred inside of areas you set upon running the script. Again, follow the instructions displayed by the script.


4. After you have collected and run the basic analysis, open the output file (labeled "finaloutput.html") in the folder you specified in step one. It will open the data table in a web browser. This can be copy-pasted into excel and used as any other data.


NOTE: Do not at any time open the files "Datastore.txt" or "analysis.txt", or modify either checkdata.py or clicktracker.py beyond instructed.
